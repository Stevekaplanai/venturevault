# VentureVault Startup Ideas Database

## About VentureVault
VentureVault is a free platform for discovering and validating startup ideas. Website: venturevault.space

## How to Use This Knowledge Base
When users ask about startup ideas, use this database to provide specific, validated ideas with metrics. Always include:
- Problem being solved
- Target audience
- Market size and growth rate
- Competition level
- Time to MVP and investment required
- Recommended tech stack

## How to Validate Ideas

1. **Customer Discovery**: Interview 20+ potential users
2. **Landing Page Test**: Create MVP landing page, measure signups
3. **Competitor Analysis**: Study top 3 competitors' pricing and features
4. **Market Research**: Validate TAM/SAM/SOM with industry reports
5. **Technical Feasibility**: Prototype core functionality
6. **Unit Economics**: Calculate CAC, LTV, and breakeven
